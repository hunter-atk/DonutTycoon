# Donut Tycoon

In this assessment, you'll build a RESTful, database-driven, HTTP server, using Express and Knex, to manage your migrated and seeded database.  You'll building this app using TDD!

## Plot

Congratulations! Your name was found in the will of a deceased donut tycoon.  Apparently she had no next of kin, and your name out of the first result on google when she searched 'booyah', and placed you as the heir to all of her donut shops before she passed.  Now the fate of 'dozens' of donut shops lies in your capable coding hands.  You must create an app that will allow you to manage your donut shops and keep track of supplies. Donut let us down!

**Time:** One Day

**Success Criteria:**
 - Build CRUD apps in Express
 - Access data using an ORM / Knex
 - Write tests in a TDD style using Mocha Chai
 - Design and implement schemas
 - Access data in datastores through joins
 - Design and implement a RESTful API
